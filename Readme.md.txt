# 🚀 ProductX – SaaS Landing Page

A fully responsive and interactive landing page for showcasing a fictional SaaS product, built with HTML, CSS, and JavaScript. Perfect for portfolio display, freelance clients, and showcasing front-end skills.

## ✨ Features

- ✅ Responsive layout (mobile-friendly)
- 🎯 CTA buttons with hover effects
- 🔧 Feature highlights with icons
- 💳 Pricing cards with styled plans
- 💬 Testimonials with profile previews
- ❓ FAQ accordion section
- 🌙 Light/Dark theme toggle
- 📩 Contact form (Formspree integration)
- 📊 Dashboard preview section
- 🔗 Live GitHub Pages hosting-ready

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Variables)
- Vanilla JavaScript (DOM manipulation)
- Google Fonts & Material Icons
- Formspree (for contact form)
